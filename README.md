# Bioinformatics-Portfolio
A curated collection of practical bioinformatics projects demonstrating end-to-end analysis of genomic and transcriptomic data using industry-relevant tools and workflows.

Objective: To apply computational methods for analyzing biological data, interpret results in a biological context, and build reproducible bioinformatics pipelines aligned with industry standards.


**Tools & Technologies**
Python (Pandas, NumPy, Biopython, Matplotlib)
R (DESeq2, EdgeR, Limma, Bioconductor packages)
Bash / Linux
FastQC, BWA, Bowtie2
GATK
IGV
Conda Environment Management
